# British Columbia (DataBC) (data-gov-bc-ca)

British Columbia (DataBC) is a open data portal open-data portal for Canada running **CKAN**, exposing a standardized catalog API over ~3,346 datasets.

**URL:** [Visit APIs.json URL](https://raw.githubusercontent.com/api-evangelist/data-gov-bc-ca/refs/heads/main/apis.yml)
